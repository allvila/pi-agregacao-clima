# API de Agregação de Dados Climáticos e Geográficos

Esta é uma API REST desenvolvida em Node.js com Express que consome APIs públicas externas (Brasil API e Open-Meteo) para retornar dados combinados e enriquecidos sobre cidades brasileiras.

## 🛠️ Tecnologias Utilizadas
- **Node.js** com **Express** (Servidor e rotas)
- **Axios** (Consumo das APIs externas)
- **Jest** e **Supertest** (Testes automatizados)
